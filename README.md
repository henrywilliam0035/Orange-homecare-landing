## ğŸš€ Getting Started

### Prerequisites

- A GitHub account
- Git installed on your computer (optional, can use GitHub web interface)
- A text editor (VS Code, Sublime Text, etc.) - optional

### Installation

1. **Create a new repository on GitHub:**
   - Go to [GitHub](https://github.com)
   - Click the "+" icon and select "New repository"
   - Name it (e.g., `orange-homecare-landing`)
   - Choose "Public" if you want to use GitHub Pages
   - Don't initialize with README (we already have one)

2. **Upload files to your repository:**
   
   **Option A: Using GitHub Web Interface (Easiest)**
   - Click "uploading an existing file"
   - Drag and drop all files:
     - `index.html`
     - `styles.css`
     - `script.js`
     - `README.md`
   - Commit changes

   **Option B: Using Git Command Line**
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Orange Homecare landing page"
   git branch -M main
   git remote add origin https://github.com/YOUR-USERNAME/YOUR-REPO-NAME.git
   git push -u origin main
   ```

3. **Enable GitHub Pages:**
   - Go to your repository settings
   - Scroll to "Pages" section
   - Under "Source", select "main" branch
   - Click "Save"
   - Your site will be live at: `https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/`

## ğŸ› Troubleshooting

**Page not displaying on GitHub Pages?**
- Check that your repository is public
- Ensure GitHub Pages is enabled in settings
- Wait 5-10 minutes after enabling
- Verify file names are correct (index.html, not Index.html)

**Smooth scrolling not working?**
- Check browser console for JavaScript errors
- Ensure script.js is properly linked in HTML
- Verify JavaScript is enabled in browser

**Styling issues?**
- Clear browser cache (Ctrl+Shift+R or Cmd+Shift+R)
- Check styles.css is properly linked
- Inspect element in browser DevTools
